# نظام إدارة عيادة الأسنان الحديث (Next.js + Supabase)

نظام متكامل واحترافي لإدارة عيادات الأسنان، مبني باستخدام أحدث التقنيات مع دعم كامل للغة العربية (RTL) وتكامل مدمج مع الذكاء الاصطناعي عبر الـ Webhooks (n8n).

## 🚀 التقنيات المستخدمة

- **إطار العمل:** Next.js 14 (App Router)
- **قاعدة البيانات:** Supabase (PostgreSQL)
- **استايل والتصميم:** Tailwind CSS
- **الأيقونات:** Lucide React
- **الخطوط:** Cairo (Google Fonts)

---

## ⚙️ التثبيت والتشغيل

اتبع هذه الخطوات لتشغيل المشروع على جهازك:

### 1. إعداد قاعدة بيانات Supabase

1. قم بإنشاء مشروع جديد على [Supabase](https://supabase.com).
2. اذهب إلى قسم **SQL Editor**، ثم قم بنسخ ولصق الكود الموجود في الملف `supabase/migrations/001_initial_schema.sql` واضغط على **Run** لتشغيل السكربت وبناء الجداول وتفعيل الحماية (RLS).
3. (اختياري) قم بتشغيل السكربت الموجود في `supabase/seed.sql` لإدخال بيانات تجريبية باللغة العربية.

### 2. إعداد المتغيرات البيئية

في جذر المشروع (المجلد `dental-clinic`)، قم بإنشاء ملف باسم `.env.local` وأضف فيه مفاتيح Supabase الخاصة بك:

```env
NEXT_PUBLIC_SUPABASE_URL=رابط_قاعدة_البيانات_هنا
NEXT_PUBLIC_SUPABASE_ANON_KEY=مفتاح_anon_key_هنا
```

### 3. تثبيت الحزم وتشغيل المشروع

افتح سطر الأوامر في مجلد المشروع ونفذ الآتي:

```bash
# تثبيت الحزم والمكتبات
npm install

# تشغيل خادم التطوير
npm run dev
```

تفضل بزيارة [http://localhost:3000](http://localhost:3000) واستمتع بلوحة التحكم الاحترافية.

## 🤖 دمج الذكاء الاصطناعي (n8n Webhooks)

يحتوي النظام على واجهة مخصصة لإدارة الـ Webhooks. عند تفعيلها:
1. تسجيل مريض جديد سيرسل Event من نوع `new_patient`.
2. حجز الموعد يرسل `book_appointment`.
3. تأكيد الموعد يرسل `confirm_appointment`.

يمكنك وضع روابط n8n الخاصة بك في صفحة **إعدادات الذكاء الاصطناعي** واختبارها بضغطة زر.
